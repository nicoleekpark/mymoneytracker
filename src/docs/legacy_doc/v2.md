## 1. Overview

v2 expands HoH Finance Tracker from a single-user, offline-first app into a **shared financial system** for families and co-owners, while adding intelligent insights through AI.

---

## 2. Family & Multi-User Support

### Family Creation
- Organizer creates a family group
- No email required
- Organizer receives a unique invite link

### Joining Flow
- Members join via invite link
- Organizer must approve join requests
- Maximum of 6 members per family
- Each member sets a username only

---

## 3. Permissions & Visibility (RBAC)

- Role-based access control
- Organizer controls:
  - Which accounts a member can see
  - Which transactions a member can see
  - Whether a member can edit or view only
- Benchmark inspiration: 1Password

---

## 4. Family vs Personal Views

- Each user has:
  - Personal dashboard
  - Family-aggregated dashboard
- Dashboards differ in:
  - Metrics shown
  - Account visibility
  - Net worth calculations

---

## 5. AI Insights

### Fixed Cost Detection
- AI identifies recurring expenses:
  - Rent
  - Utilities
  - Subscriptions
- Monthly fixed vs variable cost breakdown

### Spending Insights
- Pattern recognition
- Budget recommendations
- Month-over-month trend analysis
